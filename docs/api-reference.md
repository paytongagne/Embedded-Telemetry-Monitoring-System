# API Reference

The API is implemented with FastAPI and is designed around JSON telemetry workflows. Run the service locally and open `/docs` to inspect the interactive OpenAPI page.

## Health Check

```http
GET /health
```

Response:

```json
{
  "status": "ok",
  "service": "telemetry-monitor",
  "version": "0.2.0"
}
```

## Submit Telemetry

```http
POST /api/v1/telemetry
```

Request body:

```json
{
  "device_id": "node-001",
  "subsystem": "power",
  "temperature_c": 42.5,
  "voltage_v": 3.8,
  "battery_percent": 76.2,
  "signal_dbm": -62.0,
  "memory_usage_percent": 44.1,
  "uptime_seconds": 8421,
  "packet_sequence": 12
}
```

Response:

```json
{
  "accepted": true,
  "reading_id": 1,
  "device_id": "node-001",
  "status": "NORMAL",
  "health_score": 100,
  "alerts_created": 0
}
```

## Submit Batch Telemetry

```http
POST /api/v1/telemetry/batch
```

Request body:

```json
{
  "source": "demo-seed",
  "readings": [
    {
      "device_id": "node-001",
      "subsystem": "power",
      "temperature_c": 42.5,
      "voltage_v": 3.8,
      "battery_percent": 76.2,
      "signal_dbm": -62.0,
      "memory_usage_percent": 44.1,
      "uptime_seconds": 8421,
      "packet_sequence": 12
    }
  ]
}
```

Returns accepted counts, created reading IDs, alert count, and an ingestion-event ID.

## List Devices

```http
GET /api/v1/devices
```

Returns device status summaries with latest health score and active alert count.

## Latest Readings

```http
GET /api/v1/telemetry/latest?limit=25
```

Returns the latest stored telemetry records.

## Device History

```http
GET /api/v1/telemetry/history/{device_id}?limit=50
```

Returns recent telemetry for a single device.

## Alerts

```http
GET /api/v1/alerts?include_resolved=false&limit=100
```

Returns alert records generated by warning or critical telemetry classifications.

## Resolve Alert

```http
PATCH /api/v1/alerts/{alert_id}/resolve
```

Marks an active alert as resolved.

## Fleet Summary

```http
GET /api/v1/summary
```

Response:

```json
{
  "total_devices": 6,
  "normal_devices": 3,
  "warning_devices": 2,
  "critical_devices": 1,
  "offline_devices": 0,
  "active_alerts": 14,
  "readings_stored": 240
}
```
